# StockFlow - Inventory Management System

Versi modern dari aplikasi Inventory Management System berbasis ASP.NET Core MVC, Entity Framework Core, dan SQLite.

## Menjalankan aplikasi

```powershell
dotnet restore
dotnet build
dotnet run
```

Buka alamat localhost yang muncul pada terminal.

## Perubahan tampilan

- Sidebar modern dan responsif
- Dashboard dengan hero section dan grafik kesehatan stok
- Kartu statistik dengan ikon
- Daftar produk dengan status stok visual
- Kategori dalam bentuk kartu
- Riwayat transaksi yang lebih mudah dibaca
- Form dan halaman detail yang diperbarui
- Empty state dan notifikasi otomatis
